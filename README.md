# YouTube Homepage Clone

A front-end clone of the YouTube homepage built using HTML and CSS.  
This project was created by following and practicing concepts from the **SuperSimpleDev** tutorial, with the goal of understanding real-world layout and UI design.

## Features
- Fixed top navigation bar with search input and interactive icons
- Sidebar navigation with hover effects
- Responsive video grid built with CSS Grid and media queries
- Video cards displaying thumbnails, channel images, titles, view counts, and timestamps
- Notification badge positioned using relative and absolute positioning
- Hover tooltips on header icons

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)

## What I Learned
- Structuring a large webpage using semantic HTML
- Using Flexbox and CSS Grid for layout
- Creating responsive designs with media queries
- Positioning UI elements using relative and absolute positioning
- Debugging layout and alignment issues
